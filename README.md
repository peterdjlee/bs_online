# BS Online

An online implementation of the popular card game BS

Deployed [here](https://bsonline.shakilrafi.com)

- Jonathan Cai

## How to run

```bash
$ # Clone the repo
$ git clone https://github.com/peterdjlee/bs_online
$ cd bs_online

$ # Backend
$ cd backend
$ yarn
$ yarn start

$ # Do this in a separate terminal:
$ # Frontend
$ cd frontend
$ yarn
$ yarn start
```
